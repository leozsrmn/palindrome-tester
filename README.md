# palindrome-tester

## Usage:

├── Makefile <------------------------- Makefile of your palindrome project
├── requirement.c <-------------------- Your requirement.c
└── tests
    ├── palindrome_tester.py
    ├── requirement_tester.py
    ├── rq_tests.json
    ├── run_tests.sh 
    └── tests.json

1) cd tests/
2) chmod +x run_tests.sh
3) ./run_tests.sh
